# IntentGate 文档中心

IntentGate 是多通道卡片交互网关：统一 `CardIntent` 协议，连接 IM 客户端与智能体后端，使卡片成为可跨端复用的**交互协议**而非 LLM 输出格式。

**代码仓库路径**：`practice/intentgate/`

## 核心原则（摘要）

1. **卡片是交互协议**，智能体只产出 `CardIntent`，不拼客户端原生 JSON。
2. **Gateway 独立进程**，心跳、5 秒卡片更新、限流不得阻塞 LLM。
3. **按钮点击默认走快路径**，不触发完整 Agent 推理。
4. **Agent 后端可插拔**，AgentScope 为主，AetherMind 后期同接口接入。
5. **通道无关**，同一模板库服务企微、飞书、Web 等多端。
